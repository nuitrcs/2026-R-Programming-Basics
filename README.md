# R: Programming Basics

This repo contains materials for the 2026 intermediate R workshop from [Northwestern IT Research Computing and Data Services](https://www.it.northwestern.edu/departments/it-services-support/research/).

[How to download these materials](https://rcdsdocs.it.northwestern.edu/tutorials/git/workshop-materials.html).

[How to set up R and RStudio](https://rcdsdocs.it.northwestern.edu/tutorials/r/r-installation.html#r-install).

This workshop uses some of the tidyverse packages.  Install this metapackage if you don't already have it:

```
install.packages("tidyverse")
```

We'll also use a few other small packages that can be easily installed during the workshop.
